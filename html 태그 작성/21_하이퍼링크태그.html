a (anchor : 연결, 고정) 태그

        - 현재 HTML 문서와 다른 HTML 문서를 연결 시켜
          페이지를 이동할 수 있게 만드는 태그
        - a 태그로 감싸진 글/내용을 하이퍼텍스트로 연결(하이퍼링크를 통해서)
        --- 하이퍼링크 : hyper link (최고의 연결)

        a 태그 사용
            href(hyper reference)="연결하고자 하는 웹 사이트 주소나 html 파일 이름 작성"
            target="_blank" 새 탭에서 열기
