비디오 태그

        video 를 재생하는데 사용되는 태그
        controls : 재생, 일시정지, 음량 조절, 등의 기능을 활용할 수 있음
        autoplay : 자동재생
        muted : 비디오 음소거
        loop : 비디오가 끝나면 자동으로 반복해서 무한 재생
        source : 비디오 파일을 제공하기 위한 공간으로 실직적으로 비디오를 불러오는 공간

        ex.
        video 태그 안에 source 를 작성
        video 태그 옆에 controls 를 작성
        source 태그옆에 src 를 사용해서 불러올 위치 조정
                        type 을 작성해서 비디오 형식 지정(ex. mp4, webm)
    
      iframe : 웹에서 또 다른 html를 태그하거나 비디오를 가지고오는 태그
      비디오의 경우 저작권 문제로 외부에서 재생을 막았을 경우에는 실행할 수 없음
